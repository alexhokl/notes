Note: ある has no regular negative forms — negative conjugates as ない (see
[exceptions](#exceptions)).

- if last kana in stem form is う, あ is not used in the conjugation but わ
  instead
  * examples
    + 買う => 買わない
- exception of ある
  * non-past negative
    + ない
- te-form conjugation
  * ends with う, つ, る
    + change to って
  * ends with ぬ, ぶ, む
    + change to んで
  * ends with く
    + change to いて
    + exception
      + 行く -> 行って
  * ends with ぐ
    + change to いで
  * ends with す
    + change to して

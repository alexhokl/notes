## Chapter 1

- noun + は + noun + です。
  * これは猫です。

- noun + は + noun + ではありません。
  * これは犬ではありません。

- A: noun + は + noun + ですか。
  B: はい、noun + は + noun + です。
  B: いいえ、noun + は + noun + ではありません。
  * A: これは猫ですか。
    B: はい、これは猫です。
    B: いいえ、これは猫ではありません。

- これ/それ/あれ + は + noun + ですか。
  * これは猫ですか。

- これ/それ/あれ + は + 何ですか。
  * これは何ですか。

- noun + の + noun
  * 女の人の鞄
  * 英語の辞書
  * 何の辞書

- person + の
  * A: これは誰の辞書ですか。
    B: それは田中さんの辞書です。

- noun + は + noun + です。 noun + も + noun + です。
  noun + は + noun + ではありません。 noun + も + noun + ではありません。
  * これは辞書です。それも辞書です。
  * A: 木村さんは男の人です。森田さんも男の人ですか。
    B: はい、森田さんも男の人です。

## Chapter 2

- noun + は + い-adjective + です。
  * 富士山は高いです。
- noun + は + な-adjective + です。
  * 大坂はにぎやかです。

- noun + は + adjective + ですか、 + opposite adjective + ですか。
  * 大坂はにぎやかですか、静かですか。

- この/その/あの + noun + は + adjective + です。
  * この辞書は重いです。
- この/その/あの + noun + は + noun + です。
  * この辞書は英語の辞書です。

- ここ/そこ/あそこ + は + noun + です。
  * ここはトイレです。

- ここ/そこ/あそこ + は + 何ですか。
  * ここは何ですか。

- noun + は + どこですか。
  * お手洗いはどこですか。

- A: noun1 + は + noun2 + ですか。
  B: はい、そうです。
  B: いいえ、そうではありません。
  * A: 猫は貴方のですか。
    B: はい、そうです。
    B: いいえ、そうではありません。
  * noun2 cannot be replaced by adjective or verb

- A: noun + は + noun + です。
  B: そうですか。
  * A: この辞書は田中さんのです。
    B: そうですか。

- A: noun + は + adjective + です。
  B: そうですか。
  * A: この辞書は重いです。
    B: そうですか。

- something + ね。
  * 貴方の部屋は綺麗ですね。

## Chapter 3

- い-adjective + noun = noun
  * あれは大きい病院です。
  * この白い帽子は誰のですか。

- な-adjective + な + noun = noun
  * あれは有名な学校です。
  * あの賑やかな教室は誰の教室ですか。

- location + に + thing + が + あります
  location + に + animal + が + います
  location + に + person + が + います
  * 教室に机があります。
  * 教室に田中さんがいます。

- A + と + B
  * 筆箱の中に鉛筆と消しゴムがあります。

- A + や + B + など
  * 居間にテレビやソファーなどがあります。
  * 池の中に魚や亀などがいます。
  * など can be skipped

- location + に + 誰が + いますか。 (person)
  location + に + 何が + りますか。 (animal)
  location + に + 何が + ありますか。 (thing)
  * A: 部屋の中に誰がいますか。
    B: 田中さんがいます。

- A: location + に + something +  が + ありますか。
  B: はい、あります。
  B: いいえ、 + something + はありません。
  * A: 机の上に本がありますか。
    B: はい、あります。
    B: いいえ、本はありません。
- A: location + に + something +  が + いますか。
  B: はい、います。
  B: いいえ、 + something + はいません。
  * A: 部屋の中に人がいますか。
    B: はい、います。
    B: いいえ、人はいません。

- first question: location + に + 何がありますか。
  second question: location + には + 何がありますか。
- first question: location + に + 誰がいますか。
  second question: location + には + 誰がいますか。
  * A: 机の上に何がありますか。
    B: 本があります。
    A: 机の下には何がありますか。
    B: 机の下には鞄があります。

- A: location + に + 誰がいますか。
  B: location + には + 誰もいません。
  * A: 部屋の中に誰がいますか。
    B: 部屋の中には誰もいません。
- A: location + に + 何がいますか。
  B: location + には + 何もいません。
  * A: 机の下に何がいますか。
    B: 机の下には何もいません。
- A: location + に + 何がありますか。
  B: location + には + 何もありません。
  * A: 机の上に何がありますか。
    B: 机の上には何もありません。

- thing + は + どこに + ありますか。
  animal + は + どこに + いますか。
  person + は + どこに + いますか。
  * A: 猫はどこにいますか。
    B: 猫はベッドの下にいます。

- ... + location + ...。 location + には + ...。
  * ここは動物園です。動物園にはいろいろな動物がいます。

- どれですか。
- どの + thing + ですか。
  * どの猫ですか。
- どの人ですか。
  * 田中さんはどの人ですか。田中さんはあの人です。

## Chapter 4

- formal i-adjective in negative form
  * い => くありません
    + 新しくありません。
- casual i-adjective in negative form
  * い => くないです
    * 新しくないです。
- formal na-adjective in negative form
  * adjective + ではありません
    * 静かではありません。
- casual na-adjective in negative form
  * adjective + ではないです
    * 静かではないです。
- more casual na-adjective in negative form
  * adjective + じゃないです
    * 静かじゃないです。

- 良いです。 (いいです。)
  * 良くありません。 (よくありません。)

- te-form
  * i-adjective
    + い => くて
      + 大きくて重いです。
  * na-adjective
    + adjective + で
      + 丈夫で軽いです。

- どうですか。
  * このカレーはどうですか。
    + 美味しいです。

- あまり + adjective in nagative form
  * このカレーはあまり美味しくないです。
  * 私の部屋はあまり綺麗じゃないです。

- location + に + thing + が + quantity + ありますか。
  * 机の上に本が三冊ありますか。
  * たくさん can be used in place of quantity

- location + に + thing + が + quantity + と + another thing + が + quantity + ありますか。
  * 机の上に本が三冊と鉛筆が一本ありますか。

- いくらですか。
  * この本はいくらですか。
    + これは五百円です。

- い＜つ
  * 玩具はいくつありますか。
    + 五つあります。
  * あなたのお父さんはおいくつですか。
    + 五十歳です。

- 何 + counter
  * 学生は何人いますか。
    + 十人います。
  * 辞書は何冊ありますか。
    + 五冊あります。

- 注文
  * thing + を + ください。
    + すみません、塩をください。
    + すみません、これをください。
    + すみません、このセットをください。
  * thing + を + quantity + ください。
    + すみません、水を一杯ください。
    + すみません、このセットを二つください。
    + A: お客様、お飲み物は...
      B: コーヒーを三つください。
    + ハンパーグを一つと刺身定食を一つください。

- person + は + thing + が + 好きです。
  * A: 田中さんは何が好きですか。
    B: 私は日本料理が好きです。

- person + は + thing + が + 嫌いです。
  * A: 田中さんは何が嫌いですか。
    B: 私は魚が嫌いです。

- person + は + thing + が + しいです。
  * A: 色々な洋服がありますね。田中さんはどれがしいですか。
    B: 私はこれがしいです。

## Chapter 5

- matter + を + transitive verb
  * ご飯を食べます
  * テレビを見ます
  * お皿を洗います
  * 水を飲みます
  * 音楽を聴きます
  * 歯を磨きます
  * りんごを買います
  * 本を読みます
  * シャワーを浴びます
  * 勉強をしますお
  * 日本語の勉強をします

- anything + intransitive verb
  * 六時に起きます
  * 十一時に寝ます

- location + へ + directional intransitive verb
  * 学校へ行きます
  * うちへ帰ります
  * A: 田中さんは何時にうちへきますか。
    B: 七時に行きます。

- subject + time + action
  * 田中さんは毎晩日本語の勉強をします。
  * 田中さんは毎日日本語の勉強をします。

- location (to be entered) + に + verb
  * お風呂に入ります
  * バスに乗ります
  * 教室に入ります
  * 電車に乗ります
  * 大学に入学します
  * 病院に入院します

- location (to be exited) + を + verb
  * うちを出ます
  * 教室を出ます
  * 電車を降ります
  * 大学を卒業します
  * 病院を退院します

- ごろ
  * A: 貴方は毎朝何時にうちを出ますか。
    B: 九時ごろ出ます。

- ぐらい
  * 教室に学生が五十人ぐらいいます。
  * A: あなたは毎日何時間ぐらい寝ますか。
    B: 七時間ぐらい寝ます。

- location A + から + location B + まで + verb
  * 香港から東京まで四時間ぐらいかかります。
  * 私は毎晩八時から十時まで勉強をします。
  * 試験は十時から十二時までです。

- event + の前に + ...
  event + の後で + ...
  * 私は食事の前に手を洗います。
  * 私は食事の後で日本語の勉強をします。

- shorten answers
  * A: あなたは毎朝何を飲みますか。
    B: ~あなたは毎朝~コーヒーを飲みますか。
  * A: あなたは明日どこへ行きますか。
    B: ~あなたは明日~新宿へ行きます。
  * A: 田中さんは何曜日にここへ来ますか。
    B: ~田中さんは~月曜日に~ここへ~来ます。
  * A: あなたは授業の後で何をしますか。
    B: ~あなたは授業の後で~晩ご飯を食べます。

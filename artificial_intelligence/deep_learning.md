# Deep Learning

## Libraries

- PyTorch
  * object-oriented programming
- TensorFlow
  * object-oriented programming
  * it is developed by Google
- JAX
  * functional programming
  * it is developed by Google
- Keras
  * high-level API which can run on top of TensorFlow, PyTorch, or JAX
  * it will have support for MLX (as of 2026-01-01)
- MLX
  * a library for Apple Silicon
  * a drop-in replacement for Numpy
  * a python library
  * MIT licensed

### Un-related Libraries

- NumPy
  * a library scientific and mathematical computing
  * it runs on CPU only
  * it does not do deep learning

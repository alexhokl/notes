# HTTP

Reference: [Traceparent
Header](https://www.w3.org/TR/trace-context/#traceparent-header)

- header `traceparent`
  * contains
    + `version`
    + `trace-id`
    + `parent-id` (this can be undersood as `span-id`)
    + `trace-flags`
  * [examples](https://www.w3.org/TR/trace-context/#traceparent-header)
